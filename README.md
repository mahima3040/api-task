# API Task: Fetch Users

## Description
This Python script fetches a list of users from a public API and displays their details.

## Features
- Fetches user data using GET request
- Displays Name, Username, Email, and City
- (Bonus) Filters users whose city starts with 'S'
- Handles API errors gracefully

## How to Run
1. Install the requests library:
2. Run the script:fetch_users.py

## API Used
https://jsonplaceholder.typicode.com/users
